# RenewPilot AI

واجهة SaaS عربية باتجاه RTL لإدارة الاشتراكات والتجديدات والتنبيهات ومركز الضمان والتحليلات.

## التشغيل المحلي

```bash
npm start
```

ثم افتح:

```txt
http://127.0.0.1:3000
```

## الصفحات

- الرئيسية
- المميزات
- الأسعار
- الدعم
- تسجيل الدخول
- لوحة التحكم
- الاشتراكات
- العملاء
- التجديدات
- التنبيهات
- المركز الضماني
- التقارير
- الإعدادات

الصور المرجعية موجودة داخل `public/references`.
